Word Document Generator API

Generate DOCX documents from JSON and Word templates with schema support.

1. Getting Started

To start generating documents, you will need:

  • An OAuth2 client credentials token
  • A Word template with content controls
  • A JSON schema and instance data

2. Authentication

Use OAuth2 Client Credentials to authenticate against the API.

POST /oauth2/token
Content-Type: application/x-www-form-urlencoded

client_id=your_client_id
client_secret=your_secret
grant_type=client_credentials

3. Document Generation Endpoint

Submit a POST request to generate a document:

POST /api/documents/generate
Authorization: Bearer {access_token}
Content-Type: application/json

{
  "template": "InsurancePolicy.docx",
  "schema": { /* JSON Schema */ },
  "data": { /* Instance data */ },
  "outputFileName": "PolicyOutput.docx"
}
